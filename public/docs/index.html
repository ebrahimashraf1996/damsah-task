PostData Apis

Generate Uploading Url

If everything is okay, you'll get a 200 Success response With a Url to Upload With it.

Otherwise, the request will fail with a 500 error, and a response Error Message.

Generate Uploading Url

Example request:
$client = new \GuzzleHttp\Client();
$url = 'http://localhost:8000/api/generate-url';
$response = $client->post(
    $url,
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
            'x-api-key' => 'Secret-Api-key',
        ],
        'json' => [
            'file_name' => 'image.png',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
const url = new URL(
    "http://localhost:8000/api/generate-url"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
    "x-api-key": "Secret-Api-key",
};

let body = {
    "file_name": "image.png"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):

    "message": "Use This Url to Upload Your File"
