Introduction

This Documentation explain How to Use these Apis , These Apis relate to the task of interview in a job

Base URL: http://localhost:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

PostData Apis

PostData

Store Data into S3 and Relate it with Model Called PostData. If everything is okay, you'll get a 200 OK response.

Otherwise, the request will fail with a 500 error, and a response Error Message.

Post Data into S3

Example request:
$client = new \GuzzleHttp\Client();
$url = 'http://localhost:8000/api/post-data';
$response = $client->post(
    $url,
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
            'x-api-key' => 'Secret-Api-key',
        ],
        'json' => [
            'files' => 'files[0] image.png / files[1] video.mp4 / files[2] Doc.pdf',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
const url = new URL(
    "http://localhost:8000/api/post-data"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
    "x-api-key": "Secret-Api-key",
};

let body = {
    "files": "files[0] image.png \/ files[1] video.mp4 \/ files[2] Doc.pdf"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):


{
    "code": 200,
    "success": true,
    "message": "Uploaded Successfully"
}

Example response (500):


{
    "code": 500,
    "success": true,
    "message": "Uploading Failed"
}
